# OpenClaw Web Search Skill

A free web search skill using browser automation - no API key required!

## Installation

```bash
npx skills add hbmask/openclaw-skills@web-search
```

## Features

- 🔍 No API Key Required
- 🌐 Multi-Engine Support (Bing, DuckDuckGo)
- 📊 Structured Results
- 🔄 Auto Fallback

## Usage

```yaml
query: "your search query"
count: 5
engine: "bing"
```
